The World Wide Web (WWW), commonly known as the Web,
is the world's dominant software platform.
It is an information space where documents and other web resources
can be accessed using a web browser and (more recently) web-based applications.
The Web has changed people's lives immeasurably.
It is the primary tool billions of people use to interact on the Internet.
